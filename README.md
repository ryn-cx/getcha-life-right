# Getcha Life Right

Personal task and habit tracker. Live at **[getchaliferight.com](https://getchaliferight.com)**.

## Features

- **Tasks** with title, description, category, optional start date/time, and
  optional due date/time.
- **Recurrence**: tasks can repeat on completion or repeat on set dates.
  Intervals have precision down to the second. Monthly and yearly recurrence
  preserves the original day-of-month across short months and leap years.
- **Categories**: user-defined groups for tasks.
- **Calendar view**: full-month grid. Tasks render as bars from start to due,
  stacked into lanes when they overlap, with start/end times shown at the
  edges.
- **List view**: sortable, paginated table with a filter sidebar (status, due
  date presence, start date presence, started/not-started, category).
- **Completions**: every completion is recorded; history is viewable
  per-task and across all tasks.
- **Dark mode.**
